To run simulations:
0) Download and install the Hybrid Equations Toolbox:
    https://www.mathworks.com/matlabcentral/fileexchange/41372-hybrid-equations-toolbox-v2-04
1) Open and run initialization.m
2) Open and run the Simulink file HyEQsimulator_boostconverter_w_hysteresis_w_delays.slx
3) Open and run plotter.m to plot solutions
